# Simple-CS-island

一个使用C#制作的简易灵动岛，非常Q弹的动画。
## 功能特性

- **灵动岛交互**：鼠标悬停展开。
- **音乐播放**：按Alt后拖入音频
- **搜索**：必应搜索，现在在改Bug
- **系统托盘**：托盘中退出程序
- **网络/蓝牙**：连接自动提示
-  **复制链接**：自动提示打开

## 技术栈

- **.NET 10.0**
